## Context

Viya TMS currently runs on MongoDB Atlas. This research explores:
- Migration paths to self-hosted PostgreSQL
- Schema transformation strategies for document-to-relational conversion
- Infrastructure options for HA PostgreSQL on Kubernetes
- Risk assessment and phased migration approach
