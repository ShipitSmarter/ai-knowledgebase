# AI Commit Attribution

Research and tooling for tracking AI involvement in git commits.

## Purpose

Establish transparent attribution for AI-assisted code contributions using:
- Git trailers (structured commit message metadata)
- Environment variables for session tracking
- Git hooks for automation
- Git notes for extended metadata
