# Skill Usage Tracking

This file tracks skill usage to help understand which skills are most valuable and identify opportunities for improvement.

## How to Track Usage

When you load and use a skill, add an entry to the appropriate skill section below using this format:

```markdown
| Date | Project | Task Summary | Helpful? |
|------|---------|--------------|----------|
| YYYY-MM-DD | project-name | Brief task description | Yes/No/Partial |
```

**Guidelines:**
- Add entries after completing a task where a skill was used
- "Helpful?" indicates whether the skill provided useful guidance
- Add notes about what could be improved if applicable
