# Sendcloud

**Category:** Multi-carrier Shipping Platform  
**HQ:** Eindhoven, Netherlands  
**Founded:** 2012  
**Scale:** 30,000+ merchants, 400+ employees, €150M funding raised

## Target Market

| Segment | Description |
|---------|-------------|
| **Primary** | SMB to mid-market e-commerce |
| **Industries** | D2C brands, retail, marketplaces, 3PL |
| **Geography** | Europe (NL, BE, DE, FR, ES, IT, UK, AT) |

## Product Portfolio

| Module | Description |
|--------|-------------|
| **Shipping** | Multi-carrier label generation, shipping rules automation |
| **Pick & Pack** | Warehouse order processing, barcode scanning |
| **Tracking** | Branded tracking pages, multi-channel notifications (Email, SMS, WhatsApp) |
| **Returns** | Self-service return portal, QR-code returns |
| **Checkout** | Dynamic delivery options at checkout with ETA |

### Advanced Features (Higher Tiers)
- **Shipping Intelligence** - Carrier performance analytics, benchmarking
- **Support Automation** - Automated carrier claims handling
- **Exception Management** - Proactive delivery issue detection

## Pricing

**Model:** Subscription + per-label fee

| Tier | Monthly | Per Label | Max Labels/Month |
|------|---------|-----------|------------------|
| **Free** | €0 | - | Unlimited (limited features) |
| **Lite** | €33 | €0.11 | 400 |
| **Growth** | €99 | €0.10 | 1,000 |
| **Premium** | €195 | €0.09 | 10,000 |
| **Pro** | €799 | €0.07 | 30,000 |
| **Enterprise** | Custom | Custom | Custom |

- Overage fee: €0.15/label beyond limit
- Annual discount: ~20% off monthly price

## Key Differentiators

1. **Freemium model** - Strong self-service entry point for SMB
2. **Pre-negotiated rates** - Discounted carrier rates without volume commitments
3. **160+ carriers** - Broad European coverage
4. **E-commerce native** - Deep Shopify, WooCommerce, Magento integrations
5. **CX focus** - Branded tracking, returns, review collection

## Competitive Observations

- **Origin story** - Founded by e-commerce merchants solving their own shipping pain
- **SMB strength** - Consumer-grade UX, freemium model drives volume
- **European focus** - Deep in EU, limited outside Europe
- **Parcel only** - Not a full TMS, no freight/LTL capabilities
- **Recent AI push** - Shipping Intelligence, Support Automation features
- **Acquisition activity** - Bought Tracey, Lox, Isendu (2023-2025)
