# e2open

**Category:** Enterprise Supply Chain Platform  
**HQ:** Addison, Texas, USA  
**Founded:** 2000 (JV of Hitachi, IBM, LG, Toshiba, others)  
**Status:** Public (NYSE: ETWO) - Acquired by WiseTech Global Aug 2025 for $2.1B  
**Scale:** 480,000+ partners, 16B+ transactions/year

## Target Market

| Segment | Description |
|---------|-------------|
| **Primary** | Large enterprise with global supply chains |
| **Industries** | High tech, automotive, aerospace, CPG, pharma, retail |
| **Geography** | Global (US, EU, APAC offices) |

## Product Portfolio

e2open offers an **end-to-end supply chain platform** spanning 5 suites:

### Logistics Suite (TMS)

| Product | Description |
|---------|-------------|
| **Transportation Management** | Multimodal TMS (ocean, air, road, rail, parcel) |
| **INTTRA** | Largest ocean shipping booking network |
| **Container Management** | Via Avantida acquisition |
| **Shipment Tracking** | Real-time visibility |
| **Logistics as a Service** | Managed services (tech + people) |

### Other Suites

| Suite | Description |
|-------|-------------|
| **Planning** | Demand planning, supply planning, inventory optimization |
| **Global Trade** | Customs compliance, denied party screening, duty management |
| **Supply** | Direct materials sourcing, supplier collaboration |
| **Channel** | Channel data management, partner collaboration |

### Key Acquisitions
- **INTTRA** (2018) - Ocean shipping platform
- **Amber Road** (2019) - Global trade management  
- **BluJay Solutions** (2021, $1.7B) - Logistics execution

## Pricing

**Model:** Enterprise SaaS (not publicly disclosed)

- Annual subscription contracts
- Pricing based on transaction volume, users, modules
- Professional services for implementation
- No self-service or published pricing

## Key Differentiators

1. **Broadest platform scope** - Planning to execution in one platform
2. **Massive network** - 480K partners, 16B transactions = data advantage
3. **Ocean shipping leadership** - INTTRA is dominant booking network
4. **Multimodal TMS** - Single platform for all transport modes
5. **Control tower** - Real-time visibility, AI-powered insights
6. **ESG tracking** - Carbon footprint and sustainability reporting

## Notable Customers

HP, L'Oreal, Johnson Controls, Orange, Nutrabolt

Primarily Fortune 500/Global 2000 companies with complex global supply chains.

## Competitive Observations

- **Different segment** - Competes with SAP, Oracle, Blue Yonder for large enterprise
- **Full supply chain** - Far beyond shipping/TMS into planning, trade, sourcing
- **Ocean strength** - INTTRA gives unique ocean booking capabilities
- **High complexity** - Long implementation cycles, enterprise pricing
- **WiseTech acquisition** - Combined with CargoWise creates logistics software giant
- **Potential partner** - More likely integration partner than direct competitor for mid-market
