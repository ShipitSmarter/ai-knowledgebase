# AI Agents

This directory contains AI agent configurations, setups, and documentation for various AI agents used in our projects.

## Purpose

Store and organize:
- Agent configurations and settings
- Custom agent instructions and prompts
- Agent behavior documentation
- Integration guides for different AI agents
- Best practices for agent usage

## Structure

Each agent should have its own subdirectory or markdown file with:
- Configuration details
- Purpose and use cases
- Setup instructions
- Example interactions
- Performance notes

## Adding a New Agent

Use the template provided in `agent-template.md` to create a new agent configuration.

## Examples

- Custom coding agents
- Research agents
- Documentation agents
- QA and testing agents
- Code review agents
